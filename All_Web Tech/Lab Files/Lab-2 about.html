What is this Experiment?

This experiment demonstrates the use of HTML and CSS in designing a front page. Various CSS properties such as background-color, font-style, text-transform, display, and box-sizing are shown in practice.

Why CSS?

CSS (Cascading Style Sheets) is used to style and design webpages. It separates content from design, making it easier to manage large websites with consistent looks.
